# English-to-French-Seq2Seq-Translator-TensorFlow-Keras-with-Attention-
A fully commented English-to-French machine translation project built using a Seq2Seq Encoder-Decoder LSTM model with Attention. Includes data preprocessing, vocabulary building, training with teacher forcing, inference decoder for step-by-step translation, and saved encoder/decoder models. Ideal for NLP beginners.
